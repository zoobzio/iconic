# Integrations

Bridges between iconic and host frameworks. Each integration depends only on the public [`iconic`](../packages/iconic) package — never on the internal `@iconic/*` subpackages — so core internals stay free to change.

| Integration              | Directory           | Description                  |
| ------------------------ | ------------------- | ---------------------------- |
| [`@iconic/nuxt`](./nuxt) | `integrations/nuxt` | Nuxt module for icon sprites |
